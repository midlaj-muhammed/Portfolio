## 📦 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/midlaj-muhammed/Portfolio.git
cd Portfolio/frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
