# Midlaj | AI Engineer & Full Stack Developer Portfolio

> **"Architecting intelligent systems by combining deep learning expertise with high-performance full-stack engineering."**

A premium, high-performance personal portfolio website built with modern web technologies. This project showcases my journey as an AI Engineer and Full Stack Developer, highlighting my projects, skills, and experience through an immersive, interactive user interface.

- **Premium UI/UX**: Designed with a futuristic, "glassmorphism" aesthetic, featuring deep gradients, mesh backgrounds, and subtle glowing effects.
- **Advanced Animations**: Powered by **Framer Motion** for seamless page transitions, scroll-triggered reveals, and interactive elements.
- **Smooth Scrolling**: Integrated **Lenis** for buttery-smooth, inertial scrolling that enhances the premium feel.
- **Responsive Design**: Fully responsive layout that adapts perfectly from large desktop monitors to mobile devices.
- **AI-Driven Showcase**: Dedicated sections for AI/ML projects with live links, GitHub repositories, and tech stack details.
- **Performance Optimized**: Built on **Next.js 14 (App Router)** for lightning-fast page loads and SEO optimization.

## ğŸ“¦ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/midlaj-muhammed/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the site.
